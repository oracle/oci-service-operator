# Primary Service-Manager Layout Matrix

This matrix is the coordination source for `oci-service-operator-198.1`,
`oci-service-operator-bk6.1`, and `oci-service-operator-4ol`.

The shared `pkg/servicemanager/generatedruntime` package remains the supported
helper for generated runtime scaffolds. Do not introduce service-local
`generatedruntime` alias packages for the targets below.

For groups without a checked-in `cmd/manager/<group>` wrapper yet, keep the
monolithic `main.go` registration path as the live entrypoint. Pending
`cmd/manager/<group>` rows below are preserved manual seams, not permission to
skip `internal/registrations/<group>_generated.go` or the root manager wiring.
